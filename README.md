# Проект: Страница регистрации с валидацией


## Запуск проекта

1. Установите зависимости:
npm install
2. Запустите приложение:
npm start
3. Откройте в браузере: [http://localhost:3000](http://localhost:3000)


## Запуск тестов

1. Выполните команду
npm test

## Структура проекта

- `src/components/RegistrationForm.tsx` — форма регистрации.

- `src/api/auth.ts` — API-заглушка.

- `src/components/index.ts` — реэкспорт компонентов.

- `src/components/RegistrationForm.test.tsx` — тесты.

- `src/components/Header.tsx` — Header с навигацией
